# Adonis Fullstack Boilerplate

To install it, run:

```shell
# NPM:
adonis new yardstick --blueprint lffg/adonis-fullstack-boilerplate

# Or Yarn:
adonis new yardstick --blueprint lffg/adonis-fullstack-boilerplate --yarn
```

### Contents

- Gulp (for front-end assets compiling);
- Some view globals.
